# Notes App

A modern and responsive Notes App built using HTML, CSS, and JavaScript with a creative dashboard-style interface.

---

## Features

- Create Notes
- Delete Notes
- Search Notes
- Dark / Light Mode
- Responsive Dashboard Layout
- Local Storage Support
- Beautiful UI Design
- Modern Sidebar Navigation
- Smooth Animations
- Mobile Friendly

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome
